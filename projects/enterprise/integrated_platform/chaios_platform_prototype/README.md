## Features

- ✅ Basic Ui
- ✅ Api Endpoints
- ✅ Local Storage

## Getting Started

1. Install dependencies: `npm install` (frontend) / `npm install` (backend)
2. Start backend: `cd backend && npm start`
3. Start frontend: `cd frontend && npm start`
4. Open http://localhost:4200
